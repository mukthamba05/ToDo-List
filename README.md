
# 🚀 To-Do List App

**Manage your tasks effortlessly with this interactive To-Do List app!**  
Add, edit, delete, and track your tasks in one simple interface.

## ✨ Features

- ✅ Add new tasks instantly  
- ✏️ Edit existing tasks easily  
- ❌ Delete tasks you no longer need  
- 🎯 Mark tasks as completed  

## 🛠 How to Use

1. Open `index.html` in your browser.  
2. Type your task in the input box.  
3. Click `Add` to create a new task.  
4. Use the ✏️ icon to edit tasks.  
5. Click ❌ to remove tasks.  
6. Mark tasks as done to track progress.
